# `@mobileai/web`

`@mobileai/web` provides the embedded web binding for MobileAI:
- DOM-first `PlatformAdapter`
- React web/chat host bindings
- zone-based screen interventions
- shared rich block support (`ProductCard`, `FactCard`, `ActionCard`, `ComparisonCard`, `FormCard`)
- `useAction`, `useData`, and `useAI` hooks

This package ships the shared web API surface used by first-party web host apps.
